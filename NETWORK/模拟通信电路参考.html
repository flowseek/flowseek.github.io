模拟通信电路参考

全差分运放,：差分放大器 SC7516KAKUMY

模拟开关: BL1557是一款双通道高带宽单刀双掷(SPDT)

巴伦: 射频变压器 RFT4042A 电容C117

引脚功能对照表：

引脚名称 类型 功能描述 连接方向
ATBT_0P 芯片输出 发送信号正端 芯片 → 天线
ATBT_0N 芯片输出 发送信号负端 芯片 → 天线
ATBR_0P 芯片输入 接收信号正端 天线 → 芯片
ATBR_0N 芯片输入 接收信号负端 天线 → 芯片
ATB_0P 天线接口 天线差分正端（双向） 连接所有保护电路
ATB_0N 天线接口 天线差分负端（双向） 连接所有保护电路
LD_EN_N 切换信号 模拟开关切换 连接模拟开关

关键设计特点：

  1. 全差分设计：所有信号路径均为差分传输，提高抗干扰能力
  2. 收发分离：
    • 发送使用专用巴伦EBA16
    • 接收使用专用巴伦EBA4和射频变压器RFT4042A
  3. 三重保护：
    • 共模抑制（共模电感）
    • 电压钳位（BAT54SWT）
    • 浪涌保护（TVS）
  4. 阻抗匹配：通过巴伦和射频变压器实现芯片与天线间的阻抗转换

注意：实际应用中，TX/RX可能通过开关切换实现半双工通信，图中省略了控制逻辑部分。天线保护电路直接连接在ATB_0P/N上，同时保护收发路径。

横向布局改进说明：

  1. 信号流向优化：

    • 从左到右清晰展示信号路径：芯片 → TX路径 → 保护电路 → 天线
    • 接收路径从天线开始向右流向芯片：天线 → RX路径 → 芯片

  2. 关键元件排列：

  3. 保护电路位置：

    • 共模电感、钳位二极管、TVS管集中放置在TX路径末端
    • 直接连接天线引脚ATB_0P/N
    • 所有保护元件通过虚线连接至GND

  4. 接收路径修正：

    • 修正了芯片RX引脚的信号方向（ATBR_0P/N为输入引脚）
    • 明确RX信号流向：天线 → 射频变压器 → RX巴伦 → 运放 → 芯片

  5. 接地表示：

    • 使用虚线连接保护电路到GND节点
    • GND节点使用灰色背景突出显示

信号路径总结：

路径 信号流向 经过的主要元件
发送(TX) ATBT_0P/N → ATB_0P/N 差分放大器 → 模拟开关 → TX巴伦(EBA16) → 保护电路
接收(RX) ATB_0P/N → ATBR_0P/N 射频变压器(RFT4042A) → RX巴伦(EBA4) → 运算放大器
保护 ATB_0P/N → GND 共模电感 → 钳位二极管 → TVS管

这种横向布局更符合电路原理图的阅读习惯，清晰展示了从芯片到天线的完整信号路径，同时突出了保护电路在天线接口处的关键作用。

共模电感滤除共模干扰，
TVS防浪涌，
BAT54SWT为钳位二极管，
EBA16,EBA4为巴伦变压器，
SGM4517为模拟开关，
SGM8062为运放。
