fpgaеҠ йҖҹtensorflowжҺЁзҗҶж–№жЎҲ

дёӢйқўз»ҷеҮәеҮ з§Қдё»жөҒзҡ„ FPGA еҠ йҖҹ TensorFlow жҺЁзҗҶзҡ„ж–№жЎҲе’ҢжҖқи·ҜпјҢдҫӣдҪ еҸӮиҖғпјҡ

1. еҲ©з”Ё FPGA дё“з”ЁеҠ йҖҹеҷЁпјҲеҰӮ Xilinx DPU / Vitis AIпјү

Xilinx жҺЁеҮәзҡ„ Vitis AI е·Ҙе…·й“ҫиғҪеӨҹе°Ҷ TensorFlow жЁЎеһӢиҪ¬жҚўдёәйҖӮз”ЁдәҺ FPGA зҡ„еҠ йҖҹеҷЁ IPпјҲDPUпјүпјҢд»ҺиҖҢеңЁ Xilinx FPGA дёҠй«ҳж•ҲжҺЁзҗҶгҖӮиҜҘж–№жЎҲзҡ„дё»иҰҒжөҒзЁӢеҢ…жӢ¬пјҡ

  • дҪҝз”Ё Vitis AI зҡ„жЁЎеһӢзј–иҜ‘еҷЁе°Ҷз»ҸиҝҮи®ӯз»ғзҡ„ TensorFlow жЁЎеһӢиҪ¬жҚўдёәдёӯй—ҙиЎЁзӨәпјҲдҫӢеҰӮ DPU еҸҜиҜҶеҲ«зҡ„ж јејҸпјүгҖӮ
  • еңЁ Vitis AI ејҖеҸ‘зҺҜеўғдёӯеҜ№жЁЎеһӢиҝӣиЎҢйҮҸеҢ–пјҲеҰӮ int8пјүе’ҢдјҳеҢ–пјҢд»ҘйҖӮй…Қ FPGA зҡ„иө„жәҗдёҺеёҰе®ҪиҰҒжұӮгҖӮ
  • е°Ҷз”ҹжҲҗзҡ„ DPU IP йғЁзҪІеҲ° FPGAпјҲдҫӢеҰӮ Xilinx Alveo зі»еҲ—жҲ–еҹәдәҺ Zynq зҡ„е№іеҸ°пјүдёҠпјҢйҖҡиҝҮзЎ¬д»¶еҠ йҖҹиҺ·еҫ—дҪҺе»¶иҝҹе’Ңй«ҳиғҪж•Ҳзҡ„жҺЁзҗҶжҖ§иғҪгҖӮ
    иҝҷз§Қж–№жі•йҖӮз”ЁдәҺдјҒдёҡзә§еә”з”ЁпјҢе·Ҙе…·й“ҫжҲҗзҶҹгҖҒз”ҹжҖҒе®Ңе–„пјҢдё”иғҪж— зјқеҜ№жҺҘ TensorFlow жЁЎеһӢгҖӮ

2. еҹәдәҺ PYNQ зҡ„ FPGA еҠ йҖҹж–№жЎҲ

PYNQ жҳҜ Xilinx жҺЁеҮәзҡ„ејҖжәҗе№іеҸ°пјҢйҖҡиҝҮ Python жҺҘеҸЈи®©ејҖеҸ‘иҖ…еҸҜд»Ҙеҝ«йҖҹдёҠжүӢ FPGA ејҖеҸ‘гҖӮеҲ©з”Ё PYNQпјҢеҸҜд»Ҙе°Ҷ TensorFlow йғЁеҲҶпјҲдҫӢеҰӮеүҚз«Ҝж•°жҚ®йў„еӨ„зҗҶжҲ–йғЁеҲҶз®—еӯҗпјүеҠ йҖҹеҲ° FPGA дёҠпјҢеҗҢж—¶еҲ©з”Ёе…¶жҳ“з”Ёзҡ„ Python API еҝ«йҖҹејҖеҸ‘еҺҹеһӢгҖӮ
иҝҷз§Қж–№жі•йҖӮеҗҲеҺҹеһӢйӘҢиҜҒе’Ңе°Ҹжү№йҮҸеә”з”ЁпјҢйҷҚдҪҺдәҶ FPGA ејҖеҸ‘зҡ„й—Ёж§ӣгҖӮ

3. еҲ©з”Ё OpenNNA зӯүејҖжәҗйЎ№зӣ®

GitHub дёҠзҡ„ејҖжәҗйЎ№зӣ® OpenNNA зӯүпјҢжҸҗдҫӣдәҶдёҖж•ҙеҘ—еҹәдәҺ FPGA зҡ„зҘһз»ҸзҪ‘з»ңеҠ йҖҹеҷЁи®ҫи®ЎпјҢж”ҜжҢҒеёёи§Ғзҡ„еҚ·з§ҜгҖҒе…ЁиҝһжҺҘгҖҒжҝҖжҙ»зӯүж“ҚдҪңгҖӮйҖҡиҝҮиҝҷдәӣйЎ№зӣ®пјҢдҪ еҸҜд»ҘеңЁ FPGA дёҠе®һзҺ° TensorFlow жЁЎеһӢдёӯеҗ„з®—еӯҗзҡ„зЎ¬д»¶еҠ йҖҹпјҢе№¶дёҺ TensorFlow иҝҗиЎҢж—¶еҜ№жҺҘпјҢд»ҺиҖҢе®һзҺ°й«ҳж•ҲжҺЁзҗҶгҖӮ
иҝҷз§Қж–№жі•йҖӮз”ЁдәҺжғіе®ҡеҲ¶иҮӘе·ұеҠ йҖҹеҷЁжһ¶жһ„гҖҒдјҳеҢ–з»ҶзІ’еәҰжҖ§иғҪзҡ„ејҖеҸ‘иҖ…гҖӮ

4. зӣҙжҺҘе°Ҷ TensorFlow жЁЎеһӢиҪ¬жҚўдёә FPGA еҸҜжү§иЎҢж јејҸ

иҝ‘е№ҙжқҘд№ҹжңүдёҖдәӣеӯҰжңҜе·ҘдҪңжҺўзҙўеҰӮдҪ•вҖңйҖҸжҳҺвҖқең°е°Ҷ TensorFlow жЁЎеһӢиҪ¬жҚўе№¶жҳ е°„еҲ° FPGA дёҠпјҢдҫӢеҰӮи®әж–ҮгҖҠTransparent FPGA Acceleration with TensorFlowгҖӢжҸҗеҮәзҡ„е·Ҙе…·жөҒпјҢеҸҜд»ҘеңЁ TensorFlow еүҚз«ҜдҝқжҢҒй«ҳеұӮжҺҘеҸЈдёҚеҸҳпјҢиҖҢеңЁеҗҺеҸ°еҲ©з”Ё FPGA иҝӣиЎҢеҠ йҖҹгҖӮиҝҷзұ»ж–№жЎҲйҖҡеёёз»“еҗҲдәҶ OpenCLгҖҒHLSпјҲй«ҳеұӮж¬Ўз»јеҗҲпјүзӯүжҠҖжңҜпјҢе®һзҺ°з®—еӯҗзә§еҲ«зҡ„зЎ¬д»¶еҠ йҖҹпјҢе№¶йҡҗи—ҸдәҶзЎ¬д»¶з»ҶиҠӮпјҢи®©ејҖеҸ‘иҖ…еҸҜд»ҘеңЁзҶҹжӮүзҡ„ TensorFlow зҺҜеўғдёӢдҪҝз”Ё FPGA еҠ йҖҹжҺЁзҗҶгҖӮ
оҲҖciteоҲӮturn2academia10оҲҒ

е°Ҹз»“

жҖ»зҡ„жқҘиҜҙпјҢзӣ®еүҚ FPGA еҠ йҖҹ TensorFlow жҺЁзҗҶдё»иҰҒжңүдёӨеӨ§жҖқи·Ҝпјҡ

  • е·Ҙдёҡзә§е·Ҙе…·й“ҫпјҡеҲ©з”Ё Xilinx Vitis AI зӯүжҲҗзҶҹж–№жЎҲпјҢе°Ҷ TensorFlow жЁЎеһӢиҪ¬жҚўдёә DPU/IPпјҢеңЁ FPGA дёҠе®һзҺ°й«ҳж•ҲгҖҒдҪҺеҠҹиҖ—жҺЁзҗҶпјӣ
  • ејҖжәҗ/з ”з©¶ж–№жЎҲпјҡеҲ©з”Ё PYNQгҖҒOpenNNA жҲ–еӯҰжңҜе·Ҙе…·жөҒпјҢе®һзҺ°жӣҙеҠ зҒөжҙ»зҡ„е®ҡеҲ¶еҢ–еҠ йҖҹпјҢйҖӮеҗҲеҺҹеһӢйӘҢиҜҒе’Ңе°Ҹжү№йҮҸйғЁзҪІгҖӮ

йҖүжӢ©е“Әз§Қж–№жЎҲдё»иҰҒеҸ–еҶідәҺдҪ зҡ„еә”з”ЁеңәжҷҜгҖҒзЎ¬д»¶е№іеҸ°д»ҘеҸҠеҜ№ејҖеҸ‘е‘Ёжңҹе’ҢзҒөжҙ»жҖ§зҡ„иҰҒжұӮгҖӮеҜ№дәҺеӨ§йғЁеҲҶдјҒдёҡзә§еә”з”ЁпјҢйҮҮз”Ё Vitis AI зӯүжҲҗзҶҹе·Ҙе…·й“ҫжҳҜзӣ®еүҚдё»жөҒзҡ„йҖүжӢ©пјҢиҖҢеҜ№дәҺеӯҰжңҜз ”з©¶жҲ–еҺҹеһӢејҖеҸ‘пјҢPYNQ е’ҢејҖжәҗйЎ№зӣ®еҲҷжҸҗдҫӣдәҶжӣҙеӨҡжҺўзҙўз©әй—ҙгҖӮ

еёҢжңӣиҝҷдәӣдҝЎжҒҜиғҪеё®еҠ©дҪ зҗҶи§Ј FPGA еҠ йҖҹ TensorFlow жҺЁзҗҶзҡ„ж–№жЎҲеҸҠе…¶е®һзҺ°жҖқи·ҜпјҒ

Vitis AI е’ҢеҹәдәҺ HLS зҡ„ж–№жЎҲпјҲдҫӢеҰӮеҲ©з”Ё hls4mlпјүйғҪжҳҜеңЁ FPGA дёҠеҠ йҖҹж·ұеәҰеӯҰд№ жҺЁзҗҶзҡ„дё»жөҒж–№жі•пјҢдҪҶе®ғд»¬еңЁе·Ҙе…·й“ҫгҖҒзӣ®ж Үеә”з”Ёе’ҢејҖеҸ‘жөҒзЁӢдёҠеӯҳеңЁиҫғеӨ§е·®ејӮпјҡ

1. е·Ҙе…·й“ҫдёҺејҖеҸ‘жөҒзЁӢ

  • Vitis AI

    • йӣҶжҲҗеәҰй«ҳпјҡVitis AI жҳҜ Xilinx е®ҳж–№жҺЁеҮәзҡ„дёҖж•ҙеҘ—е·Ҙе…·й“ҫпјҢж¶өзӣ–жЁЎеһӢйҮҸеҢ–гҖҒзј–иҜ‘гҖҒDPU й…ҚзҪ®з”ҹжҲҗд»ҘеҸҠйғЁзҪІгҖӮж•ҙдёӘжөҒзЁӢеҸҜд»Ҙж— зјқеҜ№жҺҘ TensorFlowгҖҒCaffeгҖҒMXNet зӯүж·ұеәҰеӯҰд№ жЎҶжһ¶гҖӮ
    • иҮӘеҠЁеҢ–зЁӢеәҰй«ҳпјҡе·Ҙе…·й“ҫиҮӘеҠЁе®ҢжҲҗжЁЎеһӢиҪ¬жҚўе’ҢдјҳеҢ–пјҢеӨ§еӨ§йҷҚдҪҺдәҶејҖеҸ‘иҖ…зҡ„й—Ёж§ӣпјҢйҖӮеҗҲе·Ҙдёҡзә§еӨ§и§„жЁЎеӨҚжқӮжЁЎеһӢзҡ„йғЁзҪІгҖӮ
    • е•Ҷдёҡж”ҜжҢҒдёҺз”ҹжҖҒзі»з»ҹпјҡдҫқжүҳ Xilinx ејәеӨ§зҡ„зЎ¬д»¶з”ҹжҖҒзі»з»ҹпјҢжҸҗдҫӣжҲҗзҶҹзҡ„жҠҖжңҜж”ҜжҢҒе’Ңж–ҮжЎЈпјҢйҖӮз”ЁдәҺдјҒдёҡзә§еә”з”ЁгҖӮ
      оҲҖciteоҲӮturn2search6оҲҒ

  • HLS ж–№жЎҲпјҲеҰӮ hls4mlпјү

    • е®ҡеҲ¶зҒөжҙ»пјҡHLSпјҲй«ҳеұӮж¬Ўз»јеҗҲпјүж–№жі•е…Ғи®ёејҖеҸ‘иҖ…дҪҝз”Ё C/C++ жҲ– OpenCL зј–еҶҷе®ҡеҲ¶еҢ–зҡ„зЎ¬д»¶жҸҸиҝ°д»Јз ҒпјҢд»Һз»ҸиҝҮи®ӯз»ғзҡ„жЁЎеһӢпјҲйҖҡеёёжҳҜиҫғе°ҸжҲ–еүӘжһқеҗҺзҡ„зҪ‘з»ңпјүиҮӘеҠЁз”ҹжҲҗ FPGA еҸҜз»јеҗҲзҡ„еҠ йҖҹеҷЁгҖӮ
    • ејҖеҸ‘иҮӘз”ұеәҰй«ҳпјҡејҖжәҗе·Ҙе…·еҰӮ hls4ml дё»иҰҒйқўеҗ‘еӯҰжңҜе’ҢеҺҹеһӢејҖеҸ‘еңәжҷҜпјҢиғҪеӨҹй’ҲеҜ№дҪҺе»¶иҝҹгҖҒдҪҺеҠҹиҖ—зҡ„е®һж—¶еә”з”ЁиҝӣиЎҢдјҳеҢ–пјҢдҪҶйҖҡеёёйңҖиҰҒиҫғејәзҡ„ FPGA и®ҫи®ЎиғҢжҷҜгҖӮ
    • иҪ»йҮҸзә§еә”з”Ёпјҡз”ұдәҺ HLS ж–№жЎҲйҖӮеҗҲиҫғе°ҸжЁЎеһӢпјҢеӣ жӯӨеңЁйңҖиҰҒжһҒдҪҺе»¶иҝҹзҡ„еңәжҷҜпјҲдҫӢеҰӮе®һж—¶и§ҰеҸ‘зі»з»ҹпјүдёӢиЎЁзҺ°дјҳејӮпјҢдҪҶеңЁеӨ§и§„жЁЎеӨҚжқӮзҪ‘з»ңдёҠеҸҜиғҪйңҖиҰҒйўқеӨ–зҡ„е®ҡеҲ¶еҢ–и®ҫи®ЎгҖӮ
      пјҲhls4ml жҳҜејҖжәҗзӨҫеҢәжҺЁеҠЁзҡ„йЎ№зӣ®пјҢйҖӮеҗҲеҝ«йҖҹеҺҹеһӢйӘҢиҜҒгҖӮпјү

2. жЁЎеһӢи§„жЁЎдёҺйҖӮз”ЁеңәжҷҜ

  • Vitis AIпјҡ

    • ж”ҜжҢҒиҫғеӨ§гҖҒеӨҚжқӮзҡ„зҪ‘з»ңпјҲеҰӮ MobileNetгҖҒResNet зӯүпјүпјҢйҖӮз”ЁдәҺиҫ№зјҳжңҚеҠЎеҷЁе’Ңж•°жҚ®дёӯеҝғзә§еҲ«зҡ„еә”з”ЁгҖӮ
    • йҖҡиҝҮиҮӘеҠЁйҮҸеҢ–е’ҢдјҳеҢ–пјҢиғҪеңЁй«ҳз«Ҝ FPGAпјҲеҰӮ Xilinx Alveo зі»еҲ—пјүдёҠиҫҫеҲ°й«ҳеҗһеҗҗйҮҸе’ҢиҫғдҪҺеҠҹиҖ—гҖӮ

  • HLS ж–№жЎҲпјҡ

    • жӣҙйҖӮеҗҲиҪ»йҮҸзә§зҪ‘з»ңжҲ–з»ҸиҝҮеүӘжһқдјҳеҢ–еҗҺзҡ„жЁЎеһӢпјҢйҖӮз”ЁдәҺеҜ№е»¶иҝҹиҰҒжұӮжһҒй«ҳдё” FPGA иө„жәҗжңүйҷҗзҡ„еңәжҷҜпјҲеҰӮеөҢе…ҘејҸе®һж—¶еӨ„зҗҶпјүгҖӮ
    • з”ұдәҺз”ҹжҲҗзҡ„зЎ¬д»¶е®һзҺ°й«ҳеәҰе®ҡеҲ¶пјҢйҖҡеёёеҸҜд»ҘиҫҫеҲ°йқһеёёдҪҺзҡ„ж—¶е»¶пјҢдҪҶеҜ№еӨ§и§„жЁЎзҪ‘з»ңзҡ„жү©еұ•жҖ§еҸҜиғҪеҸ—йҷҗгҖӮ

3. жҖ§иғҪгҖҒе»¶иҝҹдёҺиө„жәҗеҲ©з”Ё

  • Vitis AIпјҡ

    • еҖҹеҠ©дё“з”Ё DPU е’ҢиҮӘеҠЁеҢ–е·Ҙе…·й“ҫпјҢеҸҜеңЁиҫғй«ҳж—¶й’ҹйў‘зҺҮдёӢиҝҗиЎҢеӨҚжқӮзҪ‘з»ңпјҢе®һзҺ°й«ҳеҗһеҗҗйҮҸпјҢйҖӮеҗҲжү№йҮҸжҺЁзҗҶд»»еҠЎгҖӮ
    • е·Ҙе…·й“ҫеҶ…зҪ®зҡ„йҮҸеҢ–е’ҢдјҳеҢ–зӯ–з•ҘиғҪеӨҹжңүж•ҲйҷҚдҪҺеҠҹиҖ—пјҢйҖӮз”ЁдәҺиҰҒжұӮй«ҳжҖ§иғҪдёҺиғҪж•Ҳзҡ„е·Ҙдёҡеә”з”ЁгҖӮ

  • HLS ж–№жЎҲпјҡ

    • йҖҡиҝҮе®ҡеҲ¶ HLS и®ҫи®ЎпјҢзЎ¬д»¶е®һзҺ°еҸҜд»Ҙй’ҲеҜ№зү№е®ҡзҪ‘з»ңиҝӣиЎҢж·ұеәҰдјҳеҢ–пјҢд»ҺиҖҢиҺ·еҫ—жһҒдҪҺзҡ„е»¶иҝҹпјҢзү№еҲ«йҖӮеҗҲеҚ•дёӘжҹҘиҜўжҲ–дҪҺжү№еӨ„зҗҶйҮҸеңәжҷҜгҖӮ
    • з”ұдәҺејҖеҸ‘йңҖиҰҒжүӢеҠЁи°ғдјҳпјҢиө„жәҗеҲ©з”Ёж•ҲзҺҮе’ҢжҖ§иғҪдёҠйҷҗдҫқиө–дәҺи®ҫи®ЎиҖ…зҡ„зЎ¬д»¶дјҳеҢ–иғҪеҠӣгҖӮ
