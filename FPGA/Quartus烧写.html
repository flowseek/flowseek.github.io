Intel® Quartus® Prime Lite Edition Design Software Version 18.1 for Windows

FPGA Quartus Prime 18.1初学者教程之创建工程-CSDN博客

1. .sof文件烧录（断电丢失）

（1）Programmer（程序员）

（2）Hardware Setup…（硬件设置）

（如无USB-Blaster[USB-0]，在Hardware Setup…中设置，检查硬件连接。）

（3）添加 .sof文件（工程文件夹下的output_files文件夹下）

（4）.sof文件烧写

勾选Program/Configure（程序配置）-> Start -> Progress（进度）到100%

2. .jic文件烧录（断电保存）

（1）生成 .jic文件

File -> Convert Programming Files…（转换编程文件）

（2）设置 .jic文件

1.Programming file type（编程文件类型） -> 选择 .jic类型

2.Configuration device（配置器件） -> 根据实际板卡情况选择

3.Mode（模式） -> Active Serial（主动串行）x4（根据实际板卡情况选择）

4.设置 .jic文件的位置和名称

5.Flash Loader（Flash 加载器） -> Add Device…（添加器件） -> 根据实际板卡情况选择

6.SOF Data -> Add file… -> 选择output_files文件夹下的 .sof文件 -> 点击 .sof文件，选择Properties（性能）-> 勾选Compression（压缩）-> OK -> Generate（生成）

3. .jic文件烧录

完成.jic文件烧录，断电保存，重启板卡，观察板卡显示。
